# 我只能祝你好运

最近考研卡尔的事情闹得沸沸扬扬，五战多次调剂，连续心碎武大、北邮、厦大、南开等高校，并在开直播时泪洒现场。

卡尔真的犯了很大的错吗？可能也就是在群里口嗨了几句，毕竟被人骂久了，心理渴望别人的认可也是十分正常的，吹牛逼口嗨也就那样。
我并不想为卡尔洗白，我只能祝他好运，只是他那流泪的样子多少触动了我。

毕竟自己也算是小镇做题家，考试贯穿了迄今绝大部分的人生，考试也困住了绝大部分的人生。如果考试失败，我们的人生就已经完全失败了。
在中国的当代，做学生是很难的，初中高中时似乎形势一片大好，上了大学才发现中计了。
学校教的东西都是那种基础却又十分落后的，甚至可以说是狗都不学的。在claude code等Agent迅速发展的今天，我相信绝大部分学校依旧不会开设相关课程来学习如何使用它。
至于Agent相关的课程，我也只能祝它好运。

当然，如果你按照学校的课程和节奏，找上世纪90年代或者本世纪初的工作是没有任何问题的。有的人说了，为什么好像啥也不会也能找到工作呢？这就是社会的托底作用了，
即便什么都不会，只要你的学历够，找个普通工作还是没有任何问题的。大多数工作想必都是0门槛且单一的。这些工作在不久的将来，如果运气不好，势必会被Agent取代。
而那些学历本就不出众，大学时代还全玩过去的同学呢？可能情况就十分困难了。

我的一个高中好朋友，今天突然告诉我说他要去深圳找工作了。他考公务员没有考上，在家已经待了很长时间。
我劝他先不要去太早，先做一下简历，boss上投一下，准备一下再去比较好，不然住宿啥的都要花不少钱。不曾想夜晚他已经睡在了火车卧铺。

可是他又能找到什么工作呢？他自己都不知道。这个社会应该也不知道。本科时的知识早已全忘，这是十分正常的。其实不忘大概率也找不到啥工作。
本科学的是大数据，可现在却不得不写一些excel的东西放到简历上。但其实他也压根不会excel。
做简历对于我来说，一直是个非常头疼的东西，更不要说像我朋友这样完全没经验的了。
如果没有项目，简历几乎是不可能占满一页的，甚至没有任何东西可以写。毕竟大学就是这么单薄，唯一能帮助你的就是教育经历上写上校名那几个字。对于出身不好的学校，
反而对它们的学生是副作用。

我的朋友，此去深圳，能待多久？我只能祝你好运。
