# 终究是结构性的问题

刷抖音吃到了一对网红情侣的瓜，王洋洋和夏美。大致看了一下，故事的全貌似乎可以建立起来。这是一对看上去非常另类的情侣，女强男弱。夏美是那种大女主人设，独立自主的新时代女性，王洋洋呢，似乎是一个软饭男，每个月靠夏美还花呗。我看到的图片中，王洋洋直播时说直播间的女生都是属于夫家（大致意思是这样，具体记不清），然后说夏美是属于自己的，是自由的，要捍卫夏美的穿衣自由等等，经典的那种现代女性主义者最喜欢的话语，小资产阶级最喜欢的情调。我看到不少的女生之前发的帖子或视频都以夏美为榜样，这也很正常，王洋洋是个比较文艺范的人，视频的文字很有煽动性，很容易让人忘记现实生活的苦难，让人试图从别人的美好幸福生活中来感动自己，对女孩子来说可能更是这样。

然而夏美并没有外表看上去的那般光鲜亮丽，她从17岁开始就拍摄那种大尺度的写真照，是非常确切的软色情，或者说就是色情。面对质疑，夏美说自己只是当时年轻受骗，可是她却拍了6年，这种理由是相当无力的。她还说：你可以责怪现在的我，但你能责怪17岁的夏美吗？并且还在直播间说自己可以（成功，赚钱），其他女孩同样也可以。其扭曲程度可见一斑。
网络是时时刻刻影响着现实生活的，网络暴力就是典型现象。她的言语充满诱惑，容易把女孩拉下水。人总是要为自己的存在找一个合理的解释的，夏美就是这样，讲述自己其实是没得选的，或者被骗的，然而，这里的最本质问题在于，她成功了，赚到钱了。因此，无论其如何解释，本质上就是因为夏美想要赚钱，但又不想工作，只想挣快钱而已。

这样就不得不谈到社会问题。这就是资本主义的罪恶，它总是千方百计地使人堕落。娼妓问题是资本主义社会的一大病症，它总是无限度地放大人的贪婪。贫富差距，生活的苦难足以击碎任何幻想，我们的生活中又总存在着那种“黑学”。就是那种使人黑化、堕落的学问，我们总能听到，做人自私一点、多算计一点、世界就是这样的、不行就是不努力、就说赢没赢等等。夏美其实都已经把自己的焚诀拿出来了，只要放弃道德束缚，就能获得金钱。现实生活中也有很多这样的例子，压榨学生的导师，欺骗消费者的商家以及万恶的资本家。

夏美得到了什么呢？她最想要的流量和金钱。但是命运的所有馈赠都标好了价格。人的尊严是很重要的，钱可以再挣，但是尊严可就没那么容易拿回了。但是一句尊严是完全不能够抹除现实的苦难的，事实是，一些风尘女，确实活得比较潇洒快活，这是不得不面对的事实。这也是必须要思考的问题，人们为什么不喜欢这种行为，因为它不劳而获，是完全依附别人的。

夏美因为自己赚到了钱，就觉得自己是大女主了。让我比较吃惊的是，抖音和小红书上的不少女性博主还是非常清醒的，纷纷出来抵制这种错误思潮。之前女性主义者常说的一句话是，男人最大的幸运在于没有捷径要走，导致他们能够获得更好的成长，女人因为可以走捷径，就放松了对自己的要求，这话确实不假。

做个天真的理想主义者真的很难，你说呢？
