# 五一
又是一年五一，然而写这篇文章时已经是54青年节了。

五一的这几天，可谓是放纵的不轻，暴饮暴食，三四十块的外卖说点就点，王者荣耀皮肤直接充（哪吒真好玩）。玩得脑子已经不正常了，正常做事情感觉毫无力气。干脆就狠狠地玩，否极泰来，触底反弹。在宿舍里过得昼夜颠倒，世界似乎一直是昏暗的，出宿舍才发现北京的春天真的好美😍。

最近刷到了不少关于恋爱、婚姻相关的视频，底下评论自然是吵成一团。男女朋友吃饭，到底是应该aa还是男生请客？

我们的爱情，究竟会变成什么样子？我不只一次说过，“谈恋爱”这个词，是相当怪异的。恋爱本就是一个完整的词语，且比“谈恋爱”要浪漫的多。一说恋爱，我就会想到，两个人红着脸看向对方的样子。“谈恋爱”则是两个人在吃饭、旅游。恋爱是不需要花钱的，它需要的是两个人相爱的心。我其实始终想不通的一个问题是，为什么“谈恋爱”需要花钱。在当今社会，问出这个问题估计没一个人绷得住。

我心底猜想的答案是：人们把谈恋爱当作的只是一种体验，因此就需要不断地出去旅游、吃饭。不然的话，这个“谈恋爱”该怎么进行下去呢？能做点啥呢？说实话，我想不到。当你在吃饭、旅游的时候，其实就是把对景色、美食的新鲜感转移到了你的对象身上而已。没有新鲜感刺激的谈恋爱在这个时代估计活不过1个月。

正是这样，人们一说到谈恋爱，就立刻想到花钱，立刻想到吃喝玩乐，无他，恋爱所真正需要的东西早已被时代抛弃。视频底下一条很经典的评论就是“穷人别谈恋爱”，这句话等同于说，穷人不配被爱，不应该被人喜欢，为什么？因为恋爱需要相爱的心。这是赤裸裸的资产阶级恋爱观，毫无疑问。它公开且无耻地对爱情投以最大程度地鄙夷和蔑视。这就是我们当代中国的部分所谓女性主义者，她们其实完全无法克服对男人的依赖，根本没有打破旧的秩序的勇气，仍然拜倒在资产阶级膝下，一方面她们自诩为女性主义者，一方面她们要求男人像过去那样供养，或者她们要求男人是比自己强的。

所以我不得不悲哀地认为，历史的惯性是如此的强，桎梏是如此的难以冲破。已经是21世纪的第26年了，世界变化快，但是很多东西依旧无法改变，就像那些女性主义者，恐怕大多数最终的归宿依然是结婚生子，她们可能并不会当家庭主妇，但是她们肯定依旧是依附于男人的，至少精神上，她们还是渴望一个“高大”的男人依靠，渴望一个能够让自己精神上无比膜拜的男人出现。

不过也不能怪她们，这世间的秩序本来就是如此地难以捉摸、难以冲破。我们心里虽然不同意规则，但是因为害怕、羞耻，总是默许很多，最终我们也成了旧秩序的卫道者，是我们改变了世界，还是世界改变了我和你。
