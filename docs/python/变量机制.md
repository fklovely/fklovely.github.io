# 变量

思来想去还是决定把变量的相关知识作为Python章节的第一篇，因为它确实非常基础但又非常的重要，是Python知识中的基石。
## 什么是变量?

<span style="color:red;">Python中的变量是一种引用</span>，我们来看一行简单的代码:

```python
a=10
```
上面这行代码做的事情是:创建一个值为10的对象，然后a引用这个对象，类似于c语言中的指针,需要注意的是变量本身是没有类型的，只有对象有类型。
```python
a=10
b=a
b=100
print(a)
```
输出的结果依然是10，因为b先指向10，然后又去指向了100，所以不会影响到a。
```python
a=[1,2,3]
b=a
b.append(4)
print(a)
```
输出的结果是`a=[1,2,3,4]`，因为b指向[1,2,3]，然后用append改变了这个对象，a自然也会受到影响。

## 可变vs不可变

Python中存在着可变对象与不可变对象，典型的可变对象有list,dict,set,不可变对象有int,str,tuple。

在上面的例子中也可以看到，执行`b=100`，不会改变a的值，int是不可变的，所以进行了重新绑定，而list则不然。

一些容易犯错误的地方:

```python
def add_item(lst):
    lst.append(4)

a = [1, 2, 3]
add_item(a)
```

执行的结果是`a=[1,2,3,4]`，因为list是可变的，传进去的又是引用，因此a的值也会发生变化。

```python
def change(x):
    x = 10

a = 5
change(a)
```

执行的结果是`a=5`，因为int是不可变对象。

## 深拷贝vs浅拷贝

深拷贝和浅拷贝的区别是一个非常容易令人迷糊的问题。但是核心其实很简单，只需要记住:<span style="color:red;">浅拷贝外层是新的，内层是共享的</span>，而深拷贝是全部都是新的。
```python
import copy
a = [1, 2, 3, 4, ['a', 'b']]  #原始对象

b = a  #赋值，传对象的引用
c = copy.copy(a)  #对象拷贝，浅拷贝
d = copy.deepcopy(a)  #对象拷贝，深拷贝

a.append(5)  #修改对象a
a[4].append('c')  #修改对象a中的['a', 'b']数组对象

print('a = ', a)
print('b = ', b)
print('c = ', c)
print('d = ', d)
```
输出结果为:
```python
a =  [1, 2, 3, 4, ['a', 'b', 'c'], 5]
b =  [1, 2, 3, 4, ['a', 'b', 'c'], 5]
c =  [1, 2, 3, 4, ['a', 'b', 'c']]
d =  [1, 2, 3, 4, ['a', 'b']]
```
当执行`a.append(5)`时，a和b共享对象，而c只共享内层，所以c不受影响，但`a[4].append('c')`修改的是内层的list，c也会一并更改。
